# MathSaber2

## Project Documentation

MathSaber2 is a mathematics-based game that helps users improve their math skills through various interactive challenges and puzzles.

### Features
- Interactive math problems
- Timed challenges
- Score tracking

### Installation
1. Clone the repository:
   ```
   git clone https://github.com/WilliamKlaassen/MathSaber2.git
   ```
2. Navigate to the project directory:
   ```
   cd MathSaber2
   ```
3. Install dependencies:
   ```
   npm install
   ```

### Usage
- To start the game, run:
   ```
   npm start
   ```
